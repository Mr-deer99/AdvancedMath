# 等差数列
## 通项公式
$$a_n=a_1+(n-1)d$$
式中
- $a_n$        数列的第n项
- $a_1$        数列首项
- $d$         公差
## 求和公式
$$q_n=\frac{n(a_1+a_n)}{2}$$
把$a_n$用$a_n=a_1+(n-1)d$代替得到
$$S_n=na_1+\frac{n(n-1)d}{2}$$

# 等比数列
## 通项公式
$$a_n = a_1 q^{n-1}$$
- $q$        公比
## 求和公式
$$S_n = \frac{a_1 (1-q^n)}{1-q}$$

# 求和方法
## 裂项
$$\frac{1}{(x-a)(x-b)}=\frac{1}{b-a}[\frac{1}{x-a}+\frac{1}{x-b}]$$
例如
$\frac{1}{(x-2)(x-4)}=-\frac{1}{2}(\frac{1}{x-2}-\frac{1}{x-4})$
可以先裂开，再配系数
